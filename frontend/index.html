How to investigate:
1. Request Source — Get evidence from a category
   Sends: {"action_type":"request_source","source_id":"fact_checks"}
   Returns: Text evidence related to the claim

2. Cross Reference — Check if evidence supports/refutes claim
   Sends: {"action_type":"cross_reference","source_id":"government_data"}
   Returns: NLI scores (entailment / contradiction / neutral)

3. Check Credibility — Look up a source's reliability
   Sends: {"action_type":"check_credibility","source_id":"bbc.com"}
   Returns: Bias rating, factual level, credibility score (0-1)

4. Submit Verdict — End episode with your conclusion
   Sends: {"action_type":"submit_verdict","verdict":"FALSE","evidence":[...],"confidence":0.8,"reasoning":"..."}
   Returns: Reward score (0.0-1.0) + breakdown
